## Other Timezones

To use it with other timezones, you can add the "gmt" argument at the end of your request.

```md
<div align=center>
<a href="https://github.com/metehantufek/profileclock">
<img src="https://testre-azure.vercel.app/?gmt=+3">
</a>
</div>
```

<div align=center>
<a href="https://github.com/metehantufek/profileclock">

</a>
</div>
